GeoLocation -- Description

With the call of the page 'GeoDetails' an OpenStreetMap with the current location is shown -- the 'GeoLocation'. For this location schedule+ calculates the times for sunrise and sunset on a daily basis.
     
If the position icons in and below the map are both 'blue', the text description below the map refers to the position of the blue icon in the map.
When schedule+ is called for the first time, this information may not match the current location of the installation.

GeoLocation -- Customization

1. Finding the current location

1a. Button on the map
A mouseClick on this button determines the location of the installation based on the IP address; the PostionsIcons changes to 'red'.
or
1b. Manual selection on the map
A mouseClick on the map selects a desired location; the PostionsIcon changes to 'green'.

2. Accept the selected location as 'GeoLocation'.

With both methods an arrow appears at the right lower edge in appropriate color bzw. , however, the location data is not yet used for the calculation of sunrise and sunset.
Only with a mouse click on the arrow schedule+ takes over this place as a new 'GeoLocation'. The description below the map then refers to this new position; the position icon changes to 'blue'.
The new 'GeoLocation' is now the basis for the calculation of the sunrise / sunset.
